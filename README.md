# bs2022-baru

## Pre-Required to run basestation

- install Nodejs
- install NPM
- using browser (chrome [`recommended`])

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
